# Conversor-De-Monedas
Proyectos de la Universidad Actividad 3 Proyecto 1 Conversor de Monedas.


Debes Instalar VisualStudio, luego debes instalar C y sus extensiones en visualstudio, para ejecutarlo desde VisualStudio debes
instalar php server para poder ejecutarlo de manera local, tambien usaremos el programa llamado MSYS que sirve para compilar de manera local, este sotware lo debes agregar en editar variables de entorno del sistema.

Paso 1: Debes crear el codigo de C
Paso 2: Debes guardar el archivo cpp
paso 3: Abrir la terminal
Paso 4: debes en la termial llamar a g++ -o (con el nombre del programa)
paso 5: Debes ejecutar el paquete de tu .exe con ./(ejemplo).exe
paso 6: probar la funciones de tu software.
